community_extractor: For a node segmentation and a corresponding network, creates a mask of the nodes that make up the largest
interconnected component.

community_extractor_excel: For a network, creates a new excel file containing the largest connected component in the network.

community_key_extractor: for a network and node segmentation, creates a mask of the nodes that makes up the largest connected
community containing the node of the inputted ID.

community_nerve: for an isolated node segment (from the above, for example), isolates the edges associated with the connected
component of question.

mother_extractor: for a node segmentation and its network, creates a mask of all nodes that exist between communities.

glom_draw: supports above script.
