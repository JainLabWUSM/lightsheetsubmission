radial_heatmap: used to create radial heatmaps of glomeruli angle data.
stacked_histo: Used to create overlapping histograms.
stats: Used for the following statistical tests and graphs -
	Welch's t test
	Violin plots (number of plots will need to be adjusted in code itself)
	Wilcoxon rank sums tests
	Kolmogorov-Smirnov tests for normality.

	Note although not used in the paper, this also allows:
	Student's t test
	Boxplots.
	Shapiro-Wilk test for normality.
	Anova.
